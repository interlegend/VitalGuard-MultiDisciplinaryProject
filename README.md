# VitalGuard 🛡️
### IoT Smart Wearable for Fall Detection & Heart Rate Monitoring

An advanced IoT wearable device powered by ESP32, MPU6050, 
and MAX30102 sensors. Delivers real-time fall detection and 
continuous heart monitoring to caregivers.

## 🌐 Live Website
[vital-guard-multi-disciplinary-proj.vercel.app](https://vital-guard-multi-disciplinary-proj.vercel.app)

## 🔧 Hardware
- ESP32 Microcontroller
- MPU6050 6-Axis Accelerometer/Gyroscope  
- MAX30102 Pulse Oximeter & Heart Rate Sensor

## 📁 Project Structure
- `src/` - React Web App
- `arduino/` - ESP32 Firmware
